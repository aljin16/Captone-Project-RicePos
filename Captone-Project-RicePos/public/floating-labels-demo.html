HTML Structure

Here's the basic HTML structure for floating label fields:

<div class="floating-field"> <input type="text" id="field-name" name="field_name" placeholder=" " required> <label for="field-name">Field Label</label> </div>

Key Points:

  • Use placeholder=" " (space) to trigger floating behavior
  • Always include for attribute on labels matching input id
  • Add required attribute for required fields
  • Use floating-field class as container
  • JavaScript automatically handles label floating and select value detection
